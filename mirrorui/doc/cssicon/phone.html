phone 手机图标


  <i class="ico ico-phone"></i>

手机样子变化太大了,老式的苹果机下方有个home键.后来又是全面屏了.思考一番无结果,就体现一个home键和手机上的app图标.

设计长度为容器的80%,长宽比例为1:0.618.使用:before作为手机机身轮廓,home键和app方块是:after和它的阴影实现.机身和按键在容器中绝对定位垂直居中.

机身轮廓就是一个长方形,上下边有一些厚度,这个是参考老式的非全面屏手机的样子.下边部分使用阴影(长方形在外),上边部分使用内阴影(在长方形内).

设计home键位于底部1.5%(容器)的距离.app图9个,从手机上方大概1/4的位置开始.home键就是:after,app图标则是它的阴影实现的.


完整样式


    .ico-phone
        position: relative
    
        &:before, &:after
            position: absolute
            left: 0
            right: 0
            margin: auto

        &:before
            top: 0
            bottom: 0
            width: calc(.8em * .618)
            height: .8em
            border: .02em solid currentColor
            border-radius: .03em
            box-shadow: 0 .04em,inset 0 .01em

        &:after
            bottom: .15em
            width: .06em
            height: .06em
            border: .01em solid currentColor
            border-radius: .02em
            box-shadow: .12em -.24em,0 -.24em,-.12em -.24em,
                        .12em -.36em,0 -.36em,-.12em -.36em,
                        .12em -.48em,0 -.48em,-.12em -.48em
