info information


  <i class="ico ico-info"></i>

外侧圆圈是i标记,圆点是:before,竖线是:after.绝对定位居中,上下位置设置圆点的top值和竖线的bottom值.

完整样式


    .ico-info
        position: relative
        border: .08em solid currentColor
        border-radius: 50%
    
        &:before, &:after
            position: absolute
            background-color: currentColor
            left: 0
            right: 0
            margin: auto

        &:before
            top: 10%
            width: .18em
            height: .18em
            border-radius: 50%

        &:after
            bottom: 15%
            width: .14em
            height: .4em
