加号


    <i class="ico ico-plus"></i>

这个图标比较简单,将i标记大小设置为1em,它就是图标占地大小了.加号由横线和竖线组成,分别使用:before和:after伪元素.i标记相对定位,伪元素绝对定位.令其垂直水平居中.

加号的横线和竖线厚度固定设置为4px.


      display: inline-block;
      width: 100px;
      height: 100px;
      
      // 横线
      :before
          width:100px;
          height:4px;

      // 竖线
      :after
          width: 4px;
          height:100px;

完整样式

  
    .ico-plus
        position: relative
    
        &:before, &:after
            position: absolute
            left: 0
            right: 0
            top: 0
            bottom: 0
            margin: auto
            background-color: currentColor

        &:before
            width: 1em
            height: 4px

        &:after
            width: 4px
            height: 1em
