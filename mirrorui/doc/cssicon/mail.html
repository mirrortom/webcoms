邮件

mail


  <i class="ico ico-mail"></i>

观察纸质邮件的特征,有一个开口,4条对角线在开口处相交.邮件图标宽度设计成图标容器的80%,宽高比例是0.618.即: width:.8em,height:.5em.

希望将邮件的4个对角线都展示出来,经过研究,将图标分为2部分,左右和上下.这个自然是用边框法画出三角形,而对角线正是4个三角形叠加的效果.

左右和上下部分,分别使用:before和:after.宽度和高度为0.8em,0.5em.绝对定位垂直居中于容器.开口的位置设置为高度的0.618处,这个调整上下边框厚度实现,大致是:top:.2em,bottom:.3em

左右部分:before,上下边框设为透明.同理上下部分:after的左右边框透明.

为了显示对角线,上下部分:after伪类的高度重新设置为0.56em,即比初始高度多出了0.06em.它的上下边框之和小于0.56em,多出的就是开口处的横线.它的左右边框略大于上下部分:before的左右边框,即, :after的左右边框(透明)面积大于:befter的左右边框(绿色)面积,那么重合后会显现4条对角线(就是透明部分).


完整样式


    .ico-mail
        position: relative
    
        &:before, &:after
            position: absolute
            left: 0
            right: 0
            top: 0
            bottom: 0
            margin: auto
            width: .8em
            height: .5em
            border-style: solid

        &:before
            border-width: .2em .3em .3em
            border-color: transparent currentColor

        &:after
            height:.56em
            border-width: .22em .32em .32em
            border-color: currentColor transparent
