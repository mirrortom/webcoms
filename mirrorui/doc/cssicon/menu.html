菜单

3条横杠的菜单图标.八卦中的乾卦也是它.3个点的菜单图标,是一种变形.


    <i class="ico ico-menu"></i>

    <i class="ico ico-menu dot"></i>

菜单图标就是在减号图标的基础上加了两个阴影,就变成3条横杠了.减号(中间横杠)的长度还是1em,厚度不再固定,设置成了.16em.

上边和下边的横杠是减号的阴影,阴影位置应该是图标的顶部和底部.但是希望阴影离图标顶底留出一点距离,借鉴了参考项目后,设置阴影离减号的距离是一个减号厚度.这样3条横杠,2条间距,一共是5条杠的厚度,.16em*5 = .8em.总高度未超过图标高度(1em).这样图标顶底留出.1em的距离.

 
      
      width: 100px;
      height: 16px;
      // 上,下横杠(阴影)位置,离减号(中间横杠)一个减号厚度的距离
      box-shadow: 0 32px #00000060,0 -32px #00000060;

3个点的形式就是改变了横杠的长度,将长度和宽度改成一样的就实现了,其它不变.

 
      
    .dot
      width: 16px;
      height: 16px;

完整样式

  
    .ico-menu
        display: inline-flex
        justify-content:center
        align-items:center

        &:before
            width: 1em
            height: .16em
            box-shadow: inset .16em .16em,0 .32em,0 -.32em
        // 3个点菜单,就是改width就可
        &.dot:before
            width: .16em
