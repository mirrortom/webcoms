user


  <i class="ico ico-user"></i>

用户图标比较抽象,分为头和上半身两个部分.圆形表示头,用:before.上半身:after,是一个圆角矩形.

头部大小设计为容器的40%,上半身长宽为容器的80%,40%.容器使用flex布局,列排布,等距离对齐.

肩部的圆角使用椭圆,水平半径为一半宽度,垂直半径为70%高度.


完整样式


    .ico-user
        display: inline-flex
        flex-direction: column
        justify-content: space-evenly
        align-items: center
    
        &:before, &:after
            background-color: currentColor

        &:before
            width: .4em
            height: .4em
            border-radius: 50%

        &:after
            width: .8em
            height: .4em
            border-radius: 50% 50% 0 0/70% 70% 0 0
