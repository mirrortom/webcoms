眼睛图标


  <i class="ico ico-eye"></i>

密码框显示*号内容,点击眼睛可看到明文.眼睛图标设计用于这种场景.

图标分为眼眶和瞳孔两个部分,眼眶是个长方形,长度为容器的80%,长宽比例 W:H = 8:5,用:before伪元素.右上角和左下角设置圆角实现眼眶的弧线,左上和右下角则是眼角的尖角.两个眼角是水平的,旋转眼眶即可.具体是角度,是将长方形的对角连线转到水平位子需要的角度,根据3角函数知识,就是 tan(W/H) 的角度.这个图标是 atan(5/8) = 32度.经过调整,设置在28度比较合适.

眼球用:after,是一个圆.设计半径为0.3em.

完整样式


    .ico-eye
        position: relative

        &:before,&:after
            position: absolute
            left: 0
            right: 0
            bottom:0
            top:0
            margin: auto

        &:before
            width: .8em
            height: .5em
            border: 1px solid currentColor
            border-radius:0 100% 0 100%/0 100% 0 100%
            transform:rotate(-28deg)
 
        &:after
            width: .3em
            height:.3em
            border-radius:50%
            background-color:currentColor
