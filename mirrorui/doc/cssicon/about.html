关于css图标

使用css3功能实现一些简易的常用的图标.

html标签用 <i> 标记,加上css的:after :before伪类,实际有3个元素可用.

一个质量很高的项目,参考搬运. https://viglino.github.io/iconicss (github) https://github.com/Viglino/iconicss

基本样式

所有元素的box-sizing:border-box


    .ico
        // 图片默认都是块级行内元素
        display:inline-block
        pointer-events:none
        // 垂直对齐,
        vertical-align:middle

        //面积 1em X 1em
        width:1em
        height:1em

        // 
        &:before,&:after
            content: ''
            display: inline-block

用法


    // 颜色和大小使用文本颜色和大小样式类设定
    <i class="ico ico-xxx"></i>
    <i class="ico ico-xxx text-xxl"></i>
    <i class="ico ico-xxx text-danger"></i>

常用规则

制作css图标用到的一些样式规则

content

:before,:after伪类的属性,可以写入符号,可做文字类的图标.


  content: 'i';

border

设置了4条边,为每一条边设置了颜色.


      width: 60px;
      height: 45px;
      border-width:10px;
      border-color:#f35325 #81bc06 #05a6f0 #ffba08;
      border-style:solid;

如果把盒子高度宽度设置为0,border不变,那么看起来就像4个三角形拼接成一个正方形.

可以将其它3边的颜色设置为透明,那么看起来就成了一个三角形.相邻的两个小三角形可以组合成一个大的.


      width: 0;
      height: 0;
      border: 15px solid transparent;
      border-top-color:#f35325
      border-top:30px solid #f35325;border-bottom:none;

调整border的width值,为4边设置不同的width.那么三角形看起来是任意的,不再是等腰三角形.

width值是边框的厚度,对应的是三角形的高,这个高是以三角形在图案中间那个点为顶点,到对应底边的高.

用这个特点结合透明其它边可以做出一些图案


      border-width: 10px 20px 30px 40px
      border-width: 40px 10px 10px 10px
      border-width: 10px 40px 10px 10px
      border-width: 10px 10px 40px 10px
      border-width: 10px 10px 10px 40px
      //
      border-width: 10px 10px 25px 25px;
      border-color: #f35325 #81bc06 transparent transparent;

border-radius

边框圆角效果,设置圆角border-radius:50% 50%,图案像一个游泳圈.三角形成了扇形.

来自MDN的解释: CSS 属性 border-radius 允许你设置元素的外边框圆角.当使用一个半径时确定一个圆形,当使用两个半径时确定一个椭圆.这个(椭)圆与边框的交集形成圆角效果.

border-radius理解

可以画出圆或者椭圆的效果

transform

旋转角度,作为一个搭配规则非常有用.比如要做一个黑白半球,把边框图案旋转45度看下效果.


      border-color: #ccc #ccc #333 #333;
      border-radius: 50%;
      transform: rotate(45deg)

rotate变换,是基于一个坐标系旋转.transform-origin:x y 这个规则可以设定旋转点位子.具体参考五角星图标说明.

常用的还有平移 transform:translate(left,top),可以让盒模型离开原来的位置,可以结合绝对定位使用.

上图是平移法实现居中,分2步实现.

先设置left和top都是50%,此时绿色方块的左上角位于容器的中心.但没居中,需要减去方块width和height的一半.

用translate(-50%,-50%)正好实现.它的-50%参照的就是绿色方块的with和height.于是,省掉了计算的麻烦.


    position: absolute;
    left: 50%;
    top: 50%;
    transform: translate(-50%,-50%);

scale缩放变换,也可以只按X或Y轴缩放.

黑色方块本来是边长100px的,使用缩放后只有50px了


    transform: scale(.5);

这个好像用处不大,如果要缩放改变大小,直接设置那么大就可以啊.

如果进行X,Y轴缩放再叠加旋转变换,可以改变图形的比例.



    transform: scaleX(.5) rotate(45deg);

上图是边长60px正方形,上左是X轴缩放0.5,变成了长方形,宽边长了30.

上图右是在左的基础上叠加45度旋转,变成了菱形,这个有点费解,长方形怎么会变成菱形.长方形旋转45度是这样的

如果是正方形旋转45度,那么是个特殊菱形(下左1,边长60).按照先X轴缩放,再旋转45度的套路,如果改变缩放比例,发现菱形水平对角线变化了.而且是按缩放比例变化的.


上图左1,边长大小为84.85px,这正好是60 * sqrt(2). 左2是25.46px, 即 84.85 * 0.3(缩放比例). 左3: 84.85px * 0.7 = 59.4px

这个套路可以将一个正方形盒模型变成一个菱形,通过设置scale的X/Y值,来改变菱形对角线的长度.

变换函数叠加,不同顺序得到的效果不同.

currentColor

来自MDN: currentColor 关键字代表原始的 color 属性的计算值.它允许让继承自属性或子元素的属性颜色属性以默认值不再继承

这个属性值可以设置, background-color: currentColor .如果图标的颜色是使用背景色的,那么这样设置后,也可以通过设置文字颜色color来改变图标颜色了.


    // 背景颜色是红色,通过设置字体颜色
    color:red;
    background-color: currentColor;

box-shadow

盒子阴影,图标实现利器啊.

例如这个微软图标.这个图案有4个部分,每个部分颜色不同,从左上红色块开始,这是盒子实体,设置了width,heigh都是1em,是一个正方形盒子.

其它3块都是box-shadow实现.这个css有5个参数,依次是: x-offset阴影偏离X正方向是值 y-offset 阴影模糊半斤 阴影大小 阴影颜色.

默认情况下,阴影大小和实体一样大.那么要实现第2个绿色块,需要将x-offset设置为一个实体的宽度值,也就是让阴影完全显示出来,由于阴影大小默认为实体大小,所以只要修改阴影颜色为绿色即可.

那么第3个和第4个是怎么来的.因为阴影可以设置多个的.还是那5个参数,但可以设置多条,每一条规则间用逗号隔开.

利用这个规则,继续设置黄块和蓝块,调整它们的位子和颜色.最后得到微软图标.




      display: inline-block;
      width: 1em;
      height: 1em;
      background-color: #f35325;
      box-shadow: 1em 0 0 0 #81bc06,1em 1em 0 0 #ffba08,0 1em 0 0 #05a6f0;

可以认为,无限多条的box-shadow可以增加图案内容,虽然box-shadow的形状跟随实体的形状,但可以通过调整位子和颜色还有大小做出比较丰富的图案.

无限多的box-shadow也可以用于堆码像素方法作图.

在计算机中,由于任何图案都是由像素组成的,每个像素就是一个点,这些点的颜色不同,位子不同,它们排列成一个矩阵,这个矩阵呈现的就是图案.

再回想这个box-shadow,如果把一个盒子实体作为一个"像素"单位,那么基于这个实体设置的无数box-shadow阴影,就可以得到无数的"像素",它们的大小/颜色和位子可设置,将这些"像素"按矩阵排列,那么自然也就是一幅图案.

用这个原理,做一些简易的像素画图标是可行的.

渐变

css3的渐变有很强大的绘图功能,正好用来做图标.

线性渐变 | 径向渐变 | 圆锥渐变

clip-path

canvas的剪切路径,css里也有.图案只显示在剪切路径范围内的部分.对于不那么规则的图形,可以用剪切路径实现.


    // 长方形4边中点的连线,形成一个菱形
    clip-path: polygon(50% 0,100% 50%,50% 100%,0 50%);
