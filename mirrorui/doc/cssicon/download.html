下载图标,箭头和下划线组成


  <i class="ico ico-download"></i>

图标分3部分,下划线,箭头三角形,箭头竖线.

下划线位于最底部,用容器的border-bottom画出.

箭头竖线,定位在容器top:20%处,水平居中.长度为40%,宽度为长度的一半.用:before实现.

箭头三角形底边与高比例为5:3,底边为容器的50%,水平居中.定位在容器top:60%处,即刚好与箭头竖线衔接.用:after画三角形法实现.


完整样式


    .ico-download
        position: relative
        border-bottom: .04em solid

        &:before,&:after
            position: absolute
            left: 50%
            transform: translateX(-50%)

        &:before
            top: .2em
            width: .2em
            height: .4em
            background-color: currentColor
 
        &:after
            top: .6em
            width: .5em
            border-style: solid
            border-width: .3em .25em 0
            border-color: currentColor transparent transparent
            border-bottom: none
