程序代码图标,2个尖括号和1个斜杠.


  <i class="ico ico-code"></i>

分为2部分,左右尖括号和斜杠.斜杠使用:before,高度为容器的80%,水平垂直居中,倾斜15度.

尖括号用:after.是一个宽高为容器90%和40%的矩形.也是水平垂直居中.

尖括号的尖点位于矩形竖直边的中点,尖括号的角度为90度.两边与水平线成45度角,与上下边相交形成等腰直角三角形.

尖括号是用:after的线性渐变画出的.使用4个渐变叠加,每个渐变画出一个边.

渐变的角度是45度,这个已知.要画出括号的一条边,需要计算出渐变位置.就是上图的CH(红线)的长度.

这容易得出,CEF是等腰直角,那么CH(高)就是 CE/sqrt(2). 如此,尖括号矩形高为0.4em, CH = 0.4em / 2 / 1.414 = 0.14em

尖括号厚度设计为0.04em,以0.14em渐变点为分界,前后各0.04/2em.所以css渐变线设为(0.12em~0.16).


完整样式


    .ico-code
        position: relative

        &:before,&:after
            position: absolute
            left: 0
            right: 0
            bottom:0
            top:0
            margin: auto

        &:before
            width: .04em
            min-width:1px
            height: .8em
            background-color:currentColor
            transform: rotate(15deg)
 
        &:after
            width: .9em
            height: .4em
            background-image:
                // 线厚度0.04em,以0.14em渐变点为分界,前后各0.04/2.
                linear-gradient(45deg,transparent .12em,currentColor .12em .16em,transparent .16em),
                linear-gradient(-45deg,transparent .12em,currentColor .12em .16em,transparent .16em),
                linear-gradient(135deg,transparent .12em,currentColor .12em .16em,transparent .16em),
                linear-gradient(-135deg,transparent .12em,currentColor .12em .16em,transparent .16em)
