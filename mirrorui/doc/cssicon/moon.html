月亮

月牙图标,月牙部分是boxshadow实现.


  <i class="ico ico-moon"></i>

  // 颜色和大小添加工具样式实现
  <i class="ico ico-moon text-xl"></i>

  <i class="ico ico-moon text-danger"></i>

月牙可以看成是一个圆的一部分,所以先做一个圆.这个用border-radius:50%实现.就是圆角边框.


      display: inline-block;
      border-radius: 50%;
      width: 1em;
      height: 1em;

边框阴影也会跟随圆角是一个圆形.设置box-shadow的x-offset,y-offset让阴影显露出来,调整x,y值改变月牙的大小和位子.

x值增大时,阴影再右侧增大,月牙变大.y值增大时,阴影向下倾斜,月牙呈现顺时针旋转


  box-shadow: 0 0;
  box-shadow: 10px 0;
  box-shadow: 10px 5px;

由于阴影在元素面积外,而阴影不占元素面积,所以月牙部分是在图标范围外的.这比较不妥.

可以改换内阴影inset,与阴影在元素面积外相反,内阴影在元素面积内.这样,月牙就在元素体积内.


  box-shadow: 0 0;
  box-shadow: inset -10px 0;
  box-shadow: inset -10px -5px;

阴影的颜色跟随color属性,大小用em单位.添加工具样式就可以改大小和颜色.完整css规则.


完整样式


    .ico-moon
        border-radius: 50%
        box-shadow: inset -.4em -.1em
