图表

统计图标


  <i class="ico ico-chart"></i>

图表有很多种,柱状图比较常见.结合一些柱状图的印象,将图标设计为体现柱状图表的3个元素,柱子,坐标轴,参考线.

坐标轴使用图标容器元素i,其实是容器的左和下边框,厚度设为容器的2%,即.02em.柱子和参考线使用绝对定位,在容器里水平居中对齐.

柱子使用:before和阴影实现,底部对齐.设计显示5个柱子,中间空心柱子是:before本身,两侧的实心柱子是阴影.柱子高度设计为.6em,阴影间隔.16em,即5个柱子平分容器宽度为6部分.柱子的高度差参考了K线图,错开排布.位于中间的:before为0.6em,从左到右依次是:0.2 0.4 0.6 0.5 0.3.

参考线用:after,是一条直线,长度为0.8em.倾斜30度.所有的柱子高度不超过参考线.柱子:before最高,接近这条线.

完整样式


    .ico-chart
        position: relative
        border: .02em solid currentColor
        border-top: none
        border-right: none
        overflow: hidden

        &:before,&:after
            position: absolute
            left: 0
            right: 0
            margin: auto

        &:before
            bottom: 0
            width: .1em
            height: .6em
            border: 1px solid currentColor
            box-shadow: -.32em .4em,-.16em .2em,.16em .1em,.32em .3em
 
        &:after
            top: .3em
            width: .8em
            height: 1px
            background-color: currentColor
            transform: rotate(-30deg)
