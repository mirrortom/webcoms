开关

switch使用自定义标记 <m-switch class="switch">用于2种状态,比如有效无|效,启用|禁用等.

默认样式是一个圆,开启时在右侧,关闭时在左侧.可以设置文字标题,1到2个字,过多不适合了.

最小宽度3em,圆的直径是1em,开关宽度是圆的3倍.如果设置文字标题,应将开关宽度,至少设置为 "文字个数*2em",比较合适.

// 要设置默认状态为"开",添加样式"checked".
<m-switch class="switch checked"></m-switch>
// 
<m-switch class="w-4em" on="启用" off="停用"></m-switch>

属性

on , off 在标签上设置,例如on="开" off="关"

checked 获取开关状态,true / false

//checked属性.开返回true,关返回false
let isOn = switch1.checked // true | false

事件

onClicked 设置点击后执行事件

// 点击后执行一个方法 self 就是 m-swtich 对象
switch1.onClicked = (self)=>{
  if(self.checked)
      msgbox.alert('状态是: 开');
}

原生实现

不使用自定义标记,使用原生标记实现的方式:

使用input(type=checkbox)和label两个标记,利用label的for属性和checkbox的id属性关联的特点,再用CSS选择器check判断checkbox选中时和未选中时,分别显示label的on/off状态.chekcbox放在label的前面,便于css选择器操作.

这种办法不用写js就能实现.使用自定义标记后,由于可以使用js了,所以很简单了

<div class="">
  <input type="checkbox" id="onoff1" />
  <label for="onoff1">点击label,for属性关联checkbox</label>
</div>
