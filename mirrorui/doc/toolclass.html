工具样式

为了方便使用,做一些单一功能的样式类.所谓"原子样式",组合使用多种原子样式达到一定效果.这种做法灵活搭配,也可以减少重复的css条目.

# 文本相关

文本超出容器宽度时,不换行,而是显示省略号.

需要给元素设置宽度width或者max-width,否则没有效果.

class="text-overflow" 文本超出时,显示省略号 

// text-overflow类设置了max-width:100%
<p class="text-overflow" style="width:260px" title="text-overflow文本超出时,显示省略号">
  class="text-overflow" 文本超出时,显示省略号
</p>

文本缩进

class="text-in" 用于一段文字开头缩进2em.表示一段文字的开始.

文本颜色

颜色使用的是tailwind css的色卡,采用了gray和8种彩色,色值50~900.

类名格式: text-{colorname}-{level} level取值: 0~9,colname取值: 'gray','red', 'orange', 'yellow', 'green', 'sky', 'blue', 'purple', 'pink'

class="text-sky-5"
class="text-red-5"
class="text-yellow-5"

# 背景颜色

背景颜色和文字颜色相同,类名格式一致,将text换成bg: bg-{colorname}-{level}

class="bg-sky-5"
class="bg-red-5"
class="bg-yellow-5"

# margin 和 padding

类名格式: mg-{dir}-{val} val取值范围:1~9, 10,15,20,..,95,100

dir取值:(t,b,l,r,tb,lr), 就是4个方向top/bottom/left/right,以及上下(tb)和左右(lr)2种组合方向

class="mg-5"
class="pd-5"

# 边框

类名格式: bd-{dir} dir取值(t,b,l,r)或省略

边框带圆角,类名格式: bdr-{dir} dir取值(tl,tr,br,bl)或省略

边框颜色,类名格式: bd-{colorname}-{level} 取值同文字颜色

class="bd bd-red-5 bdr-tr"
class="bd-l bd-t bd-sky-5 bdr-tl"

# 布局和对齐

文本内容或者子元素对齐

class="text-left"

class="text-center"

class="text-right"

有宽度的块级元素在父级容器中左右居中

class="mg-lr-auto"

经典的绝对定位实现水平垂直居中

class="mg-auto"

flex布局居中/居右

元素使用flex布局时,项目元素在轴线上的居中对齐.或者在轴的末端.

class="f-h-end"item2
// 主轴上居中/末
.f-h-center(end)
    justify-content: center(end)
// 垂直轴上居中/末
.f-v-center(end)
    align-items: center(end)
// flex 布局时,让子元素在垂直轴线上居中/末,用于子元素
.f-vs-center(end)
    align-self: center(end)

# 隐藏与显示

d-none d-block 媒体查询:针对不同大小屏幕 d - sm|md|lg|xl - none|block

例1,只在640px以下小屏幕才显示: class='d-block d-sm-none'

例2,只在640px以上屏幕才显示: class='d-none d-sm-block'

例3,只在992px以上大屏幕才显示: class='d-none d-lg-block'

例4,只在1200px以上PC大屏幕才显示: class='d-none d-xl-block'

例1 小屏幕 640px以下显示 例2 小平版 640px以上显示 例3 大平板 992px以上显示 例4 PC 1200px以上显示
