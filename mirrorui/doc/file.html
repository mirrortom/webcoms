# 使用自定义标记 <m-file>

采用将input控件透明化,覆盖在标题控件之上的办法.由一个外层DIV包含input-file控件,和一个说明标题.使用绝对定位,其中,input-file控件位于标题控件之上,并且是透明的.选择文件后,文件名会显示在标签中,多个文件会用逗号隔开.



# 属性 inputFile

// 返回input file的dom对象.可以获取选择的文件对象.
let file1 = document.getElementById('file1');
  
inputFileDom = file1.inputFile;

// 获取选中的第一个文件
let files0 = inputFileDom.files[0]

# 属性 files

// 返回input file dom的files对象
let file1 = document.getElementById('file1');
  
// 获取选中的文件数组
file1.files;

# 方法 reset()

// 重置组件,清空选择的文件
file1.reset();
