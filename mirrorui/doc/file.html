文件选择

使用自定义标记,m-file.采用将input控件透明化,覆盖在标题控件之上的办法.由一个外层DIV包含input-file控件,和一个说明标题.使用绝对定位,其中,input-file控件位于标题控件之上,并且是透明的.选择文件后,文件名会显示在标签中,多个文件会用逗号隔开.

<m-file></m-file>
// 实际会生成
<m-file class="input-file">
  <input type="file">
  <label class="form-label">请选择文件...</label>
</m-file>

属性

file.inputFile 文件控件的dom对象

// 返回input type=file的dom对象.
let file1 = document.getElementById('file1');
inputFileDom = file1.inputFile;
// 获取选中的第一个文件,也可以做其它设定
let files0 = inputFileDom.files[0]

file.files 文件控件的files对象,input(file)的原始files对象

// files对象就是input控件的files对象
let file1 = document.getElementById('file1');
// 获取选中的文件数组
file1.files;

tag 设置提示语,用tag属性

multiple 要多选文件,加multiple属性

disabled 禁用控件,加disabled属性

// 提示,多选
<m-file tag="多选文件,一次选择不要超过5个." multiple></m-file>
// 禁用
<m-file disabled></m-file>

方法

file.reset() 重置组件,清空选择的文件

file1.reset();
