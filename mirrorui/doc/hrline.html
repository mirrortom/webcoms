hr分隔线

使用hr标记,使用时改变颜色和边距以及厚度.常用于内容分割线.

hr标记的语义是画一条水平线

例子

原始的hr标记,不同浏览器可能不同.浏览器给hr的默认样式是,高度为0,边框为1px,计算出来是0.8px.

// 原始hr
<hr />
边框法和高度法实现

边框: hr设置上边框1px,高度为0,线条颜色是边框色

高度: hr设置1px高度,边框置空,线条颜色是背景色

边框型
高度型 
// 边框
<hr class="bd-none bd-1 bd-gray-3"/>
// 高度
<hr class="h-1 bd-none bg-red-5"/>

原始hr也是边框,但边框法修改的地方在于,它只有一个上边框,而原始hr有4个边框,由于高度为0,似乎是4边框合并成一条线了,但是,如果设置一个高度,它和div盒模型一样.

区别

这种方法从效果看,有一点区别的,使用高度的,比使用边框的,线条厚度要厚一些.但是,都是1px为什么效果有些差别呢?

通过查看元素发现,边框法计算的边框厚度是0.8px,而高度法计算的厚度是1px.这是浏览器计算的问题,预计和显示器像素有关系.

上下边距

添加上下边距,让分割线和内容有一定距离,这是常用做法.

样式

使用边框法,修改border-style得到不同样式的线.

<hr class="bd-none bd-t-3 bd-t-sky-3 bd-t-dotted mg-tb-30" />
