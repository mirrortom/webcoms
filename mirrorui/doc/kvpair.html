名值对

一对标签,分为标题和内容两个部分,中间有分割线.适用于展示带有标题的内容,内容简短的.

容器,标题和内容使用3个span,通常是左边标题右边内容,也可以是上面标题下面内容.

例子

左边标题,右边内容,中间适用竖线分开.和标签类似,使用背景/文字/边框设计不同风格.

Name Value 爱好 游戏 提示 水深危险 YouTube 网上电视 
<span class="inline-flex j-start a-center bg-gray-2 ">
  <span class="bd-r-1 bd-r-gray-5 pd-lr-10">Name</span>
  <span class="pd-lr-10">Value</span>
</span>

  <span class="bg-green-2 pd-lr-10 cir-tr-5 cir-br-5">游戏</span>
</span>
