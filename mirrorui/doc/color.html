颜色

颜色值采取的是tailwind css的色卡,除了黑白灰,还有11个彩色.

灰和11种彩色,选择了100-900梯度,数值越大,颜色越深.tailwind color

黑白,各附带一个相近色,以替代纯色.颜色来自tailwindcss的 Neutral-100,Neutral-900

黑-0 #000 黑-1 #171717 白-0 #fff 白-1 #f5f5f5

主题

一个暗色全局主题,用css变量实现,影响的内容包含字体背景和边框颜色.在html上添加样式theme-dark实现.

// 背景色
--back-color: #fff
// 前景色,字体颜色
--front-color: #171717
// 边框色
--border-color: #d1d5db
// 暗色主题
.theme-dark
    color-scheme: dark;
    --back-color: #171717;
    --front-color: #f5f5f5;
    --border-color: #4b5563;

组件不支持全局主题,但有彩色主题,影响范围各个组件不同,但基本上也是边框/背景/文字颜色/hover效果这些.在组件上添加颜色名字改变主题.一共有11个彩色.

// 红色按钮主题
class="btn red"
