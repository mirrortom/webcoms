信息提示

使用自定义标记 <m-msgshow> 用于在页面上显示一个提示信息,可用于操作后返回一个结果说明.

属性

title 消息标题.dom上设置属性

msg 消息内容.dom属性

show dom属性,默认是不显示的,添加show属性会显示

keep 默认2秒倒计时关闭.含有keep属性时,不关闭!

方法

show(msg, style, autoClose=true) 显示消息.使用预定义风格

show(msg, title, styles, autoClose=true) 显示消息,使用自定义风格

// msg:消息内容
// title:标题
// style:风格类型:[success,error,warning,info,notify]
// styles:自定义样式数组
// autoClose:true时,3秒自动关闭.
msgshow1.show('服务器返回成功!','success');

clear() 关闭.会清空消息内容,隐藏消息框.右侧的X按钮执行的也是这个方法.

风格

可以添加背景色或者字体色自定义风格,预定义的风格是常用的几种信息提示场景,为了方便使用.

成功Success 提示Info 错误Error 警告Warning 公告notify 自定义风格
