信息提示

用于在页面上显示一个提示信息,可用于操作后返回一个结果说明.

# 使用自定义标记 <m-msgshow> ,也可以使用div,结合样式类 .msgshow

提示 内容... 
<m-msgshow class="mg-b-15 bg-sky-4" title="信息" msg="提示内容"></m-msgshow>

// 用div时,里面的内容也需要写全
<div class="msgshow mg-b-15 bg-green-5" title="失败" msg="操作失败">
  <span class="title">提示</span>
  <span class="msg">内容...</span>
</div>

# 标记属性

title 设置标题

msg 设置提示内容

# 方法

调用js方法显示4种常用场景风格提示框,以及自定义标题和内容.

ok() info() err() warn() show() clear() 

  // 对象
  let msg1 = document.getElementById('msgshow1');

  // 成功
  msg1.ok('操作成功.');

  // 失败
  msg1.err('操作失败.');

  // 警告
  msg1.warn('非法操作.');

  // 自定义,最后一个参数是样式类,接收1个或者多个
  msg1.show(msg, title, ...style)
  msg1.show('自定义内容', '自定标题', 'text-white-1','bg-black-1');

  // 清空 内容会清空,标记会隐藏
  msg1.clear();
